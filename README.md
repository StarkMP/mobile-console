### Requirements

- [Node.js](https://nodejs.org) v.18.0.0^

### Install packages

```sh
npm install
```

### Build production

```sh
npx turbo build
```

### Run development hot-reload server

```sh
npx turbo dev
```

### Development hints

#### Pre-commit checklist

- [x] npm run format
- [x] npm run lint
- [x] Increase the version at **package.json**
- [x] Add update notes to **CHANGELOG.md**
